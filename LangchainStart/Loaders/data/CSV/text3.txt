# 🚀 KnowledgeVault AI

> AI-powered personal knowledge assistant built using **LangGraph**, **LangChain**, **MCP (Model Context Protocol)**, **Google Gmail API**, **Google Drive API**, and **Groq LLMs**.

---

## 📌 Overview

KnowledgeVault AI is a multi-MCP agent that can connect to:

* 📧 Gmail
* 📁 Google Drive
* 🧠 Analytics/Summarization Engine

The agent can retrieve information from multiple sources and generate intelligent summaries using LLMs.

---

## ✨ Features

### Gmail MCP

* Fetch recent emails
* Search emails
* Extract sender, subject, date, and snippets
* Summarize inbox activity

### Google Drive MCP

* List recent files
* Search files by name
* Retrieve file metadata
* Analyze Drive activity

### Analytics MCP

* Structured summarization
* Main topics extraction
* Important points extraction
* Action item generation
* Short summary generation

### Agent

* Multi-server MCP integration
* Tool calling with LangGraph
* Autonomous reasoning
* Cross-tool analysis

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/KnowledgeVault-AI.git

cd KnowledgeVault-AI
```

## 2️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

### Windows

```bash
.venv\Scripts\activate
```

### Linux / Mac

```bash
source .venv/bin/activate
```

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

# ☁️ Google OAuth Setup

## Step 1 — Create Google Cloud Project

Visit:

```text
https://console.cloud.google.com
```

Create a new project.

---

## Step 2 — Enable APIs

Enable:

* Gmail API
* Google Drive API

Path:

```text
APIs & Services → Library
```

---

## Step 3 — Configure OAuth Consent Screen

Navigate:

```text
APIs & Services
→ OAuth Consent Screen
```

Choose:

```text
External
```

Fill:

* App Name
* Support Email
* Developer Email

---

## Step 4 — Add Scopes

Add:

```text
https://www.googleapis.com/auth/gmail.readonly
```

```text
https://www.googleapis.com/auth/drive.readonly
```

---

## Step 5 — Add Test User

Navigate:

```text
OAuth Consent Screen
→ Test Users
```

Add your Gmail account.

---

## Step 6 — Create OAuth Credentials

Navigate:

```text
Credentials
→ Create Credentials
→ OAuth Client ID
```

Choose:

```text
Desktop Application
```

Download:

```text
credentials.json
```

Place it in the project root:

```text
KnowledgeVault-AI/
```

---

# 🔐 First Authentication

Run:

```bash
python -m GmailMCPTools.emails
```

Google login page opens.

Approve permissions.

After successful authentication:

```text
token.json
```

will be generated automatically.

# 🚨 Security Notes

Never upload these files:

```text
.env
credentials.json
token.json
```

Add them to `.gitignore`.

Example:

```gitignore
.env
credentials.json
token.json

__pycache__/
*.pyc
```

# ⚠️ Current Limitations

* Reads email metadata only
* Does not fetch full email body
* Reads Drive metadata only
* Does not read actual file contents
* Single-user OAuth flow
* Local token storage
